Crie um programa que leia nome e duas notas de vários alunos e guarde tudo em uma lista composta. No final, mostre um boletim contendo a média de cada um e permita que o usuário possa mostrar as notas de cada aluno individualmente.

1. Criar duas listas, uma temporária e outra que vai guardar todos os dados dos alunos. -
2. Utilizar o loop while para receber os dados dos alunos. -
3. Guardar os dados dentro da lista temporária. -
4. Guardar a lista temporária dentro da lista principal.
5. Limpar a lista temporária.
6. Perguntar se o usuário que adicionar mais alunos.
7. No final exibir a tabela de todos os alunos e sua média de nota.
8. Perguntar ao usuário de qual aluno ele deseja ver a nota.
9. Exibir os dados do aluno escolhido.
